# 江湖路迢迢 - 旅游推荐系统

一个现代化的旅游目的地推荐网站，帮助用户发现和探索世界各地的美丽景点。

## 项目概述

本项目是一个基于 HTML5、CSS3 和 JavaScript 的旅游推荐系统，采用响应式设计，支持桌面端和移动端访问。系统集成了丰富的交互功能，包括目的地搜索、分类筛选、用户登录、收藏管理等。

## 功能特性

### 🏠 首页功能

- **Hero 横幅**：全屏沉浸式背景图，展示旅行主题标语
- **快速导航**：一键跳转到各功能区域
- **品牌标识**："江湖路迢迢"品牌展示

### 🔍 搜索功能

- **实时搜索**：支持目的地名称、国家、描述、标签的模糊搜索
- **防抖优化**：输入停止 200ms 后才执行搜索，提升性能
- **搜索结果高亮**：点击搜索结果自动定位并高亮对应卡片
- **键盘支持**：ESC 键关闭搜索面板

### 🏷️ 分类筛选

- **分类标签**：全部、海滩、都市、自然四大类别
- **可视化图标**：使用 FontAwesome 图标直观展示分类
- **平滑过渡**：切换分类时自动滚动到目的地区域

### 🎠 热门推荐轮播

- **自动轮播**：每 2.5 秒自动切换热门目的地
- **漂浮卡片设计**：右侧展示可点击的缩略卡片
- **动态背景**：主背景图随选中卡片切换
- **指示器导航**：底部圆点指示器，支持点击切换

### 📍 目的地展示

- **网格/列表视图切换**：两种展示模式自由切换
- **分页浏览**：每页显示 8 个目的地，支持上下页导航
- **卡片信息**：展示图片、名称、描述、评分、价格等信息
- **快捷操作**：收藏按钮、地图按钮

### 💝 收藏系统

- **未登录收藏**：支持游客模式收藏（临时存储）
- **登录同步**：登录后收藏列表自动同步到用户账户
- **实时状态**：收藏按钮实时显示收藏状态

### 👤 用户系统

- **用户注册**：支持昵称、手机号、密码注册
- **用户登录**：手机号密码验证登录
- **个人中心**：查看收藏列表、订单记录
- **信息编辑**：支持修改昵称

### 📋 详情弹窗

- **目的地详情**：展示目的地完整信息
- **用户评价**：自动生成模拟评价数据
- **统计数据**：评分、评价数、游客数展示
- **操作按钮**：收藏、查看地图、探索冒险

### 🗺️ 地图功能

- **百度地图集成**：使用百度地图 API 展示位置
- **标记显示**：显示选中目的地和其他景点位置
- **图例说明**：区分选中目的地与其他景点

### ✨ 冒险模式

- **沉浸式展示**：全屏背景图轮播
- **漂浮文字效果**：优雅的文字动画
- **缩略图导航**：右下角可点击缩略图
- **详细介绍**：景点亮点和详细描述

## 运行方式

### 开发环境

1. **直接打开**：由于项目纯前端实现，无需后端服务，可直接用浏览器打开 `index.html`

2. **本地服务器（推荐）**：
   ```bash
   # 使用 Python 启动简易服务器
   python -m http.server 8000
   
   # 或使用 Node.js
   npx serve .
   ```

3. **访问地址**：`http://localhost:8000`

### 部署建议

- 可部署至任何静态文件托管服务
- 推荐使用：GitHub Pages、Netlify、Vercel、阿里云 OSS 等
- 部署时需确保 `images/` 目录完整上传

## 特色功能亮点

1. **毛玻璃效果**：导航栏和搜索框采用 backdrop-filter 实现毛玻璃效果
2. **水滴按钮**：登录/注册按钮采用水滴形状设计，带有光效动画
3. **漂浮动画**：冒险模式中的文字漂浮效果
4. **平滑滚动**：页面内导航使用平滑滚动
5. **卡片高亮**：搜索结果定位时的高亮动画
6. **图片预览**：点击画廊图片可全屏预览

## 使用说明

### 游客模式

1. 打开页面即可浏览所有目的地
2. 可使用搜索功能查找目的地
3. 支持收藏（临时存储，刷新后丢失）
4. 点击目的地卡片查看详情

### 用户登录

1. 点击右上角用户图标
2. 选择登录或注册
3. 登录后收藏列表自动保存
4. 可在个人中心查看收藏和订单

### 搜索技巧

- 输入目的地名称、国家名搜索
- 支持按标签搜索（如"海滩"、"浪漫"）
- 支持中文和拼音搜索
